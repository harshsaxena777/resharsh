# Interactive AI & Data Science Portfolio | Streamlit Web Application

A clean, modern, and production-ready interactive resume web application built using **Streamlit**, customized Python logic, and advanced **HTML5/CSS3 embedding**. This application is engineered to showcase a data-driven background in AI/ML engineering, Natural Language Processing (NLP), and Vulnerability Assessment & Penetration Testing (VAPT).

## 🌟 Key Features

- **Responsive Professional Layout:** Clean typography and multi-column grid architecture designed specifically to align with enterprise expectations (e.g., Zycus AI/US Delivery roles).
- **Interactive Sidebar Panel:** Provides seamless UX for recruitment teams to download a traditional offline PDF version of the resume directly from the cloud.
- **Optimized for ATS Parsing:** Built using semantic sections matching modern automated screening protocols.
- **Cloud-Native Framework:** Ready to be deployed instantly on Streamlit Community Cloud with minimal setup overhead.
